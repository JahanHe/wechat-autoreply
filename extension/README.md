# Chrome 扩展说明

这个目录是扩展源码。Chrome 实际加载目录由构建脚本生成：

```text
dist/wechat-kf-extension
```

构建：

```bash
npm run build-extension
```

安装和迁移请看项目根目录：

```text
PORTABLE_INSTALL.md
```

当前功能：

- 自动监听微信小店客服页新消息
- 自动打开左侧未读或更新会话
- 固定规则优先回复
- 规则未命中时调用本地 AI 服务
- AI 回复前经过审核层
- 回复按空格、逗号、顿号、换行拆成短句
- 同一会话后续消息不重复发快速承接语
- 客户不耐烦时先道歉
- 可合规引导会员专区和视频号主页产品介绍
- 严禁引导客户加微信、打电话、私聊、留手机号或私下交易
