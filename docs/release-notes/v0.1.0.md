# 小店AI客服 v0.1.0

> 第一版正式桌面安装包。这个版本的重点不是“能跑脚本”，而是把小店AI客服整理成可下载安装、可配置、可通知、可升级的桌面工作台。

## 下载安装

| 系统 | 文件 | 适合谁 |
| --- | --- | --- |
| macOS Apple Silicon | `wechat-autoreply-macos-arm64.dmg` | M 系列芯片 Mac |
| Windows 安装版 | `wechat-autoreply-windows-setup.exe` | 希望正常安装到系统的 Windows 用户 |
| Windows 便携版 | `wechat-autoreply-windows-portable.exe` | 想直接运行、不走安装流程的 Windows 用户 |

## 首次运行

打开应用后只需要补三项配置：

| 步骤 | 位置 | 说明 |
| --- | --- | --- |
| 1 | 悬浮窗 > API | 填 DeepSeek API Key |
| 2 | 悬浮窗 > 通知 | 填企业微信机器人 Webhook |
| 3 | 桌面客服页 | 微信扫码登录小店客服 |

真实 API Key 和 Webhook 不会进入仓库，也不会被打进安装包。

## 已内置能力

- 独立 Electron 桌面应用，不依赖手动打开浏览器标签页。
- 桌面悬浮窗，可查看状态、开启/暂停 Bot、配置 API、Webhook、规则库和助手风格。
- 默认文字规则、图文规则、回复图片、商品卡片和邀请下单动作。
- 商品码 `10000275472384` 已配置为“润宇年度会员商业社群”。
- 客服页需要扫码时，会通过企业微信 Webhook 通知。
- 回复失败、页面异常、AI 服务异常、图片或文件路径缺失时，会即时通知。
- 成功回复进入小时总结和每日总览，不逐条刷屏。
- Webhook 临时失败会写入补发队列，恢复后自动补发。
- GitHub Actions 自动构建 macOS DMG、Windows 安装版和 Windows 便携版。

## 安全边界

- 不要提交或转发 `.env`。
- 本版本不会绕过扫码、验证码或平台风控。
- 需要人工登录时，应用会通知你扫码。
- 严禁引导客户加微信、打电话、私聊、留手机号或私下交易。

## 已知限制

- macOS 和 Windows 安装包当前未做商业代码签名，首次打开可能出现系统安全提示。
- 素材库和快捷语动作接口已预留，但默认业务规则没有启用素材库/快捷语，避免误发。
- 如果微信小店客服页结构变化，需要重新捕捉页面结构并更新动作选择器。
