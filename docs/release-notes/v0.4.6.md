# 小店AI客服 v0.4.6

> `v0.4.6` 聚焦知识库体验、客户记忆管理、AI 参考编辑、监控看板可视化和通知图标一致性。生产回复链路继续沿用 `v0.4.5` 的本机知识索引边界。

## 下载

| 系统 | 文件 |
| --- | --- |
| macOS Apple Silicon | `xiaodian-ai-kefu-macos-arm64.dmg` |
| Windows 安装版 | `xiaodian-ai-kefu-windows-setup.exe` |
| Windows 便携版 | `xiaodian-ai-kefu-windows-portable.exe` |

## 新增

- 知识库新增“客户记忆”二级页，可查看客户历史、最近消息、客服回复、已发送规则、已执行动作和未完成任务。
- 客户记忆支持双模式压缩：本机摘要不联网，AI 摘要调用已配置的远方 AI API。
- 新增客户记忆 IPC：`main-get-customer-memories`、`main-compress-customer-memory`、`main-compress-customer-memories`。
- AI 参考页新增 Markdown 预览卡片和弹窗编辑器，长文本不再挤在小 textarea 中。
- 监控总览新增视觉状态卡、回复来源比例条和右侧原始快照详情。

## 变更

- 知识库二级结构调整为：知识总览、规则库、AI 参考、测试中心、客户记忆。
- 行内高频操作改为图标按钮：展开、复用、启停、删除、压缩、AI 压缩、复制等均带 tooltip 和无障碍标签。
- 监控页移除重复的“当前状态”和“快捷操作”板块，避免和顶部栏、悬浮窗入口重复。
- 桌面通知统一使用最新应用 logo，避免系统通知显示旧图标或默认图标。

## 修复

- 保存 AI 参考开关时不再覆盖已有回复人格、边界规则、手动知识和参考回复长文本。
- 客户记忆过长时可以手动压缩，只保留最近对话并沉淀长期摘要，减少后续 AI 上下文膨胀。
- 小窗口下客户记忆、AI 参考和监控看板自动单列排布，降低文字遮挡和横向溢出风险。

## 真实环境验收

- 在已登录微信小店客服页中验证主控台、悬浮窗和 Bot 运行状态仍同步。
- 打开知识库 > 客户记忆，确认客户列表、展开详情、本机压缩、AI 压缩和复制 ID 正常。
- 打开知识库 > AI 参考，确认 Markdown 预览、弹窗编辑、保存开关和长文本保存正常。
- 打开监控 > 总览，确认状态卡点击后右侧详情展示对应后台快照。
- 验证文字规则、图片/文件动作、商品卡、邀请下单、AI API 回复和本机知识检索能力没有回退。

## 安全说明

- API Key、Webhook、Cookie、控制 Token、客户缓存和私有同步资料不会进入仓库或安装包。
- 外部知识源仍只用于同步到本机，生产回复不实时查询外部服务器。
- 微信小店客服页映射属于第三方网页自动化场景，请仅在账号权限和平台规则范围内使用。
