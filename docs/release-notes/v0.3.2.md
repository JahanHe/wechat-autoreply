# 小店AI客服 v0.3.2

> 这一版把应用可见名称统一为“小店AI客服”，补齐长期运行设置，并保持旧版本配置可自动迁移。

## 下载文件

| 系统 | 文件 | 说明 |
| --- | --- | --- |
| macOS Apple Silicon | `wechat-autoreply-macos-arm64.dmg` | M 系列 Mac |
| Windows 安装版 | `wechat-autoreply-windows-setup.exe` | 正常安装到 Windows |
| Windows 便携版 | `wechat-autoreply-windows-portable.exe` | 免安装直接运行 |

## 主要变化

- 应用名、窗口标题、托盘提示、悬浮窗标题、扩展名称、发布标题和文档入口统一改为“小店AI客服”。
- 打包产物里的 macOS `.app` 名称改为 `小店AI客服.app`，Windows 安装应用名也同步更新。
- 运行目录改为 `~/Library/Application Support/小店AI客服/` 和 `%APPDATA%/小店AI客服/`。
- 首次启动新版时，如果旧目录 `wechat-shop-kf-bot` 存在且新目录还没有配置，会自动复制旧配置，避免规则、Webhook 和 API Key 丢失。
- Bot 接管页新增“长期运行”设置：开机自启动、守护检查、脚本无心跳重载、防后台清退/防休眠、AI/页面/脚本心跳间隔。
- 总览状态新增“长期运行”状态灯，直接显示是否开机自启、是否防休眠。
- 守护定时器改为保存后立即生效，不需要重启应用。
- 主控制台新增“初始化”页。首次缺 DeepSeek Key、企业微信 Webhook 或外部判断库 Session Cookie 时会自动打开。
- 初始化页点“保存并自检”后，会自动写入本机配置并检查 AI、Webhook、判断库 10 条测试、规则库和长期运行状态。
- 初始化自检完成后可直接切到客服页映射扫码登录。
- 源码模式下的 macOS LaunchAgent 和 Windows 任务计划名称更新，并会清理旧任务名。

## Logo 和草稿目录

安装包实际使用并提交的是：

- `desktop/assets/logo.png`
- `build/icon.png`
- `build/icon.icns`
- `build/icon.ico`

本地 `logo-concepts*` 目录是候选设计草稿，不属于运行依赖，也不会进入安装包。这样可以避免仓库和 release 体积被草稿图撑大。
