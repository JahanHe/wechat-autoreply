# 小店AI客服 v0.3.7

> 本版重点修复“客户发图片/表情/商品卡后无回复”、规则库无法现场试跑、图片动作不稳定、外部知识库调用过程不可见的问题。

## 规则和非文本消息

- 客户发送图片、表情、文件、视频等非文本内容时，不再让脚本进入“已失联/无回复”状态。
- 规则匹配会把消息类型也纳入搜索，例如 `[图片]` 会同时匹配“图片、截图、非文本、客户发图片”等别名。
- 新增默认动作规则“非文本消息：引导文字描述”，安装包首次运行和旧版本升级都会自动补齐。
- 规则库新增“手动激发测试”：可先只测试命中，也可对当前客服会话真实执行命中规则。
- 手动测试结果会展示命中来源、规则名、动作列表、执行结果和处理步骤。

## 图片、商品和异步回复

- 图片粘贴发送改为等待预览/发送按钮出现，再点击发送，并确认消息区或上传弹窗状态变化。
- 图片动作执行失败时会显示“已上传、已粘贴、已发送或失败原因”，不再把未真正发送的动作记为成功。
- 默认图文规则、商品卡规则和邀请下单规则继续随安装包内置，并由回归测试确认商品码仍可命中。
- 15 秒承接语和 60 秒兜底回复不会截断最终 AI/外部知识库补充回复。

## 外部知识库可视化

- API 测试改为 deep 模式，能同时验证本地 AI 和外部知识库链路。
- 测试结果会显示是否查询外部知识库、命中几条、本地/远端来源、思考状态、审核状态和耗时。
- 回复日志继续记录外部知识库、思考状态、审核和处理步骤，方便复盘为什么某次回复没有采用外部知识库。
