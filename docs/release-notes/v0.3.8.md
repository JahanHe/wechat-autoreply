# 小店AI客服 v0.3.8

> 本版根据结构诊断报告，优先收敛后端安全边界、规则真实执行保护、桌面导航层级和旧 CLI 入口。

## 下载

| 系统 | 文件 | 说明 |
| --- | --- | --- |
| macOS Apple Silicon | `wechat-autoreply-macos-arm64.dmg` | 拖到 Applications 后打开 |
| Windows 安装版 | `wechat-autoreply-windows-setup.exe` | 正常安装到 Windows |
| Windows 便携版 | `wechat-autoreply-windows-portable.exe` | 不安装，双击运行 |

## 后端闭环

- DeepSeek 调用改为原生 `fetch`，不再把 API Key 放进 `curl` 命令行参数。
- 本机 HTTP 控制接口增加 `DESKTOP_CONTROL_TOKEN` 鉴权，`/action`、页面捕捉和判断库刷新等操作必须带 token。
- 首次启动会自动生成本机控制 token 并写入运行目录 `.env`。
- 旧 Playwright CLI 不再作为 `npm start` 默认入口；`npm start` 改为启动桌面应用，旧入口保留为 `legacy:*`。

## 前端结构

- 左侧导航改为 5 个分组：工作台、自动回复、AI 与判断库、通知与运行、系统。
- 每个入口增加图标、短标签和 hover 说明；默认保持简洁，鼠标悬浮或当前选中时才展开介绍。
- 页面标题同步显示图标，降低纯文字界面的扫描压力。
- “真实执行命中规则”增加二次确认，执行前显示客户消息、命中规则和动作清单。

- `npm run check:secrets` 通过。
- `npm run test:notify-outbox` 通过。
