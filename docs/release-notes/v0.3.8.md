# 小店AI客服 v0.3.8（已废弃）

> 本版本保留历史说明，但不再作为推荐下载版本。`v0.3.9` 已以 `v0.3.7` 为功能基线重建，并人工保留了 v0.3.8 中验证有效的安全和生命周期修复。

## 为什么废弃

`v0.3.8` 尝试同时调整后端安全边界、规则执行保护和桌面导航结构，但界面层级和悬浮窗方案被后续验证为不适合继续沿用。为避免把复杂 UI 带入正式产品，`v0.3.9` 使用正式 `git revert` 回到 `v0.3.7` 行为基线，再重新分阶段实现安全、退出闭环、模块化和四入口主控台。

## 曾包含的方向

- DeepSeek 调用改为原生 `fetch`，避免 API Key 出现在 `curl` 命令行。
- 本机 HTTP 控制接口增加控制 Token。
- 规则真实执行增加保护。
- 导航和悬浮窗尝试过新的分组设计。

## 当前推荐

请下载和使用 `v0.3.9`：

| --- | --- |
