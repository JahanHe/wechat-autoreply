# 小店AI客服 v0.3.1

> 这是 `v0.3.0` 的安装包修复版。重点修复另一台 Mac 启动时报 `Cannot find module ... app.asar/src/runyu-judgments.js` 的问题，并把 macOS 打开失败排查说明放进 DMG。

## 下载文件

| 系统 | 文件 | 说明 |
| --- | --- | --- |
| macOS Apple Silicon | `wechat-autoreply-macos-arm64.dmg` | M 系列 Mac |
| Windows 安装版 | `wechat-autoreply-windows-setup.exe` | 正常安装到 Windows |
| Windows 便携版 | `wechat-autoreply-windows-portable.exe` | 免安装直接运行 |

## 修复内容

- 修复打包遗漏：`src/runyu-judgments.js` 现在会进入 `app.asar`，另一台 Mac 不会再因找不到外部知识库模块而崩溃。
- `server.js` 和 `desktop/main.js` 依赖的 `src/**/*` 均纳入安装包。
- `docs/**/*` 纳入安装包，方便离线查看说明。
- DMG 根目录新增 `mac-install-troubleshooting.md`，说明未签名 App、隔离属性、重新安装、扫码、图片/文件路径等常见问题。
- README 和图文指南下载链接更新到 `v0.3.1`。

## 如果你已经安装 v0.3.0

如果启动时看到：

```text
Cannot find module ... app.asar/src/runyu-judgments.js
```

请删除旧 App 并安装 `v0.3.1`：

1. 退出“小店AI客服”。
2. 删除 `/Applications/小店AI客服.app`。
3. 下载 `v0.3.1` 的 DMG。
4. 重新拖到 `Applications`。
5. 第一次打开被 macOS 拦截时，右键 App 选择“打开”，或执行：

```bash
xattr -dr com.apple.quarantine "/Applications/小店AI客服.app"
open "/Applications/小店AI客服.app"
```

## 保留 v0.3.0 的功能

- 默认动作规则：权益图文、进群图文、发商品、邀请下单。
- AI/外部知识库优先，15 秒承接列表，60 秒兜底列表。
- 同一会话避免重复发送同一句静态话术。
- 规则库图片/文件路径支持选择替换和打开位置。
- 应用图标、托盘图标和安装包图标统一。
